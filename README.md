# hybrid-aml-detection-engine

Ensemble Scoring: Merges Supervised, Unsupervised, and Rule-based logic.

Regulatory-Ready: Outputs JSON payloads compatible with India's FINnet 2.0 (FIU-IND) portal.

Adversarial-Hardened: Specifically designed to catch Smurfing (via velocity checks) and Identity Fraud (via KYC quality metrics).

Zero-Trust Alerts: Every alert is cryptographically signed using HMAC-SHA256, preventing criminals from tampering with their own risk scores during transit.
